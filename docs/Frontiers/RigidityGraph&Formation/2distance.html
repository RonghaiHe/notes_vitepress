Distance-based Rigidity Formation ​

Formation control of mobile agents based on inter-agent distance dynamics

Preliminaries ​

Graph Rigidity ​

Given an undirected graph G=(V,E),V={1,⋯,N}. Let pi∈R2 be assigned to each vertex i. A framework is defined as a pair (G,p), where p=(p1,⋯,pN)∈R2N.

The rigidity function rG:R2N→R|E| associated with (G,p) is defined as

rG:=(⋯,∥pi−pj∥2,⋯),(i,j)∈E.

where |E| denotes the cardinality of E and ∥⋅∥ denotes the 2D Euclidean norm. For simplicity, we assume that edges are ordered as ⋯,(i,j),⋯,(k,l),⋯ such that i>j,j⩾k and k>l.

Euclidean distance matrix and graph realizability ​

A matrix D=(dij)∈RN×N is a Euclidean distance matrix (EDM) if and only if there exists a point (p1,…,pN)∈RmN, where pi∈Rm,i=1,…,N, such that dij=‖pi−pj‖2 for i,j=1,…,N.

The triangular inequality holds for elements of EDMs such that dij≤dik+dkj for all distinct i,j, and k. The dimension m of the affine span of points p1,…,pN generating an EDM D is the embedding dimension of D. An EDM D is invariant under the translation and rotation of p.

Given an undirected graph G=(V,E),V={1,…,N}, a pair (G,d), where d=(…,dij,…),(i,j)∈E, is realizable if and only if there exists a point (p1,…,pN)∈RmN such that dij=‖pi−pj‖2 for i,j=1,…,N. That is, (G,d) is realizable if and only if rG−1(d) is not empty. The partial matrix D′ induced by ( G,d ) such that dij′=dij if (i,j)∈E and all the other elements are unspecified has an EDM completion D if and only if (G,d) is realizable.

Problem formulation ​

Consider an N single-integrator modeled agent group in the plane:

(1)p˙i=ui,i=1,⋯,N,

where pi∈R2 and ui∈R2 denote the position and control input, respectively, of agent i.

The sensing topology among agents of the group is modeled by an undirected graph G=(V,E), which is referred to as the sensing graph of the group. We assume that (G,p) is rigid. Each agent i measures relative displacements of its neighbors,

(2)pji:=pj−pi,j∈Ni,i∈V,

where Ni is the set of all neighbors of agent i. Due to the absence of a common directional-sense, the directional information contained in pji cannot be used in a global sense. The distance-based control laws, thus, generally adjust ∥pji∥ rather than pji, while the displacement-based control laws directly adjust pji.

Given a point p∗=(p1∗,…,pN∗)∈R2N of G such that (G,p∗) is infinitesimally rigid, the desired formation is defined as

(3)Ep:={p∈R2N:‖pi−pj‖=‖pi∗−pj∗‖,i,j∈V}.

That is, Ep is the set of all formations congruent to p∗.

Thus the overall group task is the stabilization of p to satisfy

‖pi−pj‖=‖pi∗−pj∗‖,i,j∈V

Then the subtask of each agent i can be assigned as the stabilization of its position pi to satisfy

(4)‖pi−pj‖=‖pi∗−pj∗‖,j∈Ni,i∈V

The consistency between the overall group task and the subtasks of agents is crucial, i.e., the achievement of all subtasks should imply the completion of the overall group task. The rigidity of (G,p∗) is linked to the consistency. Based on Definition of rigidity, it follows from the rigidity of (G,p∗) that there exists a neighborhood Uq of q∈Ep such that if p∈Uq, then the desired formation is achieved by satisfying (4), i.e., rG−1(rG(p∗))∩Uq⊂Ep. Note that if (G,p∗) is not rigid, then rG−1(rG(p∗))∩Uq⊄Ep.

The formation control problem is then formulated as follows:

Problem 1: Given N(N⩾3) single-integrator modeled agents, an undirected graph G=(V,E) and a point p∗∈R2N of g, suppose that (G,p∗) is infinitesimally rigid. Design a control law u∈R2N such that the desired formation Ep in (3) is asymptotically stable under the control law, based on measurements (2).
