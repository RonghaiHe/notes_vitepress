语法高亮 ​

VitePress 提供了由 Shiki 驱动的语法高亮功能，并支持额外的特性，例如行高亮：

输入

md
```js{4}
export default {
  data () {
    return {
      msg: '高亮显示！'
    }
  }
}
```

输出

js

  data () {
    return {
      msg: '高亮显示！'
    }
  }
}
